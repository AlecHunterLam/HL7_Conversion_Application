HL7 to JSON Data Convertor

This is a basic web application focused on improving interoperability in the Healthcare Information Technology industry. Due to rigid systems among Electronic Medical Record Systems in the healthcare industry, transferring data is difficult and can be time consuming and cost inefficient.

To address this issue, we developed a web application that allows users to take in raw HL7 data, and outputs the information in a json format. This allows the data to be used by individuals that work in other forms of data storage. The application helps move the healthcare industry a small step forward in improving interoperability among electronic medical record systems.
